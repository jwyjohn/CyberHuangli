祭祀：指祠堂之祭祀、即拜祭 {lang} 。
祈福：就是 GitHub 上香还愿，祈求神明降福或设醮还愿之事。
求嗣：向神明祈求后嗣(子孙)。
开光：Release 发布后、供奉上位之事。
塑绘：编写文档、画 UML 图等。
齐醮：设醮建立 {lang} 环境祈拜、求平安等事。
斋醮：配置 {lang} 环境前需举行的斋戒仪式。
沐浴：祈福设醮或还愿时清洁电脑灰尘之谓。
酬神：还愿，答谢 {lang} 神恩。
造庙：进行 Compile，Assemble，Link，Build，Release。
祀灶：亦称"祭灶"、"送灶"。即祭拜 {lang} 神。
焚香：将一些带有特殊气味和芳香气味的源代码放在火焰中烟熏火燎。
出火：谓移动神位，即变更 {lang} 环境。
嫁娶：嫁娶指的是男娶女嫁，举行结婚大典，迎亲之日。
纳采：同结婚姻，受授聘金，也就是下聘礼。俗称：订婚、文定、过订、完聘、大定。
问名：求问 Repo 、 Package 、 Class 或变量等的名字。
安床：安置睡床卧铺，一为新婚安置新床、二为 Debug 不顺重新安新床。
合帐：作蚊帐之事，今应进行一个 merge 。
订盟：订婚仪式的一种，俗称小聘(订)。也是订亲。
进人口：指收纳养子女，或认干儿子、干女儿，或进行一个 fork 。
裁衣：临近 Realeas 前一个月，程序员要备酒、蓝布和金银钗环交给媒人送到测试人员手中。
修墓：也就是修理 Deprecated 的代码。
启攒：指坟墓「洗骨」之事。俗谓「重构屎山」也。
破土：仅指埋葬用的破土，指将 Repo 进行 Archive 。
立碑：竖立墓碑或纪念碑。
开生坟：指项目未死，先找地作坟墓。
普渡：祭祀超渡阴界的老旧代码。
入宅：即迁入新 Repo 、所谓“新居落成典礼”也。
安香：安装 {lang} 之神位或 GitHub 之神位。
安门：指配置跳板机等工事。
修造：也就是写代码或改代码。
起基动土：开发时、第一次动提交 Commit。这个是除了 initial commit 以外的动土。
上梁：装上建筑物屋顶的大梁，西式开发指 Merge 重要组件。
开井开池：开凿新的存储空间，俗称买硬盘。
作陂放水：建筑蓄水池、将水灌入蓄水池，指向硬盘里装奇奇怪怪的东西。
拆卸： rm -rf
伐木做梁：复制粘贴代码。
作灶：指安修新的服务器。
解除：扫舍；指冲洗生产环境服务器、解除灾厄。
穿屏扇架：制作门扇、屏障等工作，指进行安全审计。
盖屋合脊：进行一次 Build 。
开厕： docker run
造仓库：git init
平治道涂：指修理依赖等工事。
谢土：Release 完工后所举行的祭祀。
开市：商品行号开张做生意。包括年头初V开始营业或建 Repo 或新创账号等事。
挂匾：指悬挂招牌或各种匾额，现指给 Repo 起名字。
立券交易：订立与 {lang} 的契约。
纳财：指已属于自己的财富基础上购置产业、进货、收帐、收租、讨债、贷款、五谷入仓等。
求财：相对于纳财来说求财是指未属于自己的财富而祈求通过投资等活动赚取博得财富。
开仓：出货财，出售数据集和代码。
安机：安装 Mainframe 。
造车器：建造矿机。
经络：连接网线等各类线材。
酝酿：指摸鱼等事。
出行：指外出旅行、观光游览。
移徙：指更换开发环境之意。
分居：指 Repo 分家、各自另起炉灶之意。
栽种：栽种植物，或接枝。
纳畜：招收项目小弟。
捕捉：断蚁，扑灭家中或农作物之 Bug 。
教牛马：训练 {lang} 牛马之工作。
会友：聚集朋友；有朋相会。
